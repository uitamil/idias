Features

  2. Features

Document Request

Document creator has a privilege to raise the document request to the quality user. Document request raised by the Quality user will have an auto generated “Request number” to track the request effectively by both user and creator.

Online “Document Submission system”

The system allows users to login using by a unique ID and password combination with autolock mode active. System forces the user to change the password periodically and lock the user out upon repeated login failures. Files and folders cannot be deleted / modified / copied / added whereas date and time cannot be changed by Analyst or Reviewer.

Storage and Tracking system

Document for both perpetual and non-perpetual documents will be tracked and maintained with support of “Auto Reference Number”. Documents will be stored with appropriate tracking system by chain relation system (Document – Storage racks).

Administrator

Administrator shall have the right to create / delete the access or user accounts of “Document Creator” and Quality User”.

Audit Trail Review

All entry modifications will be updated in the audit trail are yet restricted to intentional or accidental modifications. Documents will be stored with appropriate tracking system by chain relation system (Document – Storage racks). Backup storage allows anytime recovery of data in printable / PDF / Excel sheet formats. System audit trail report covers the detailed list of Login, logout, auto log off, creation, modification, deletion of records, comments, User ID creation, modification, including the date, time, and person.

Back up

Backup storage allows anytime recovery of data in printable / PDF / Excel sheet formats.

Log sheet

Documents Issued / Tracked / Archived will be monitored through “Document Log Sheet” under pre-defined period.

Security and password protection system

The system allows users to login using by a unique ID and password combination with autolock mode active. System forces the user to change the password periodically and lock the user out upon repeated login failures. Files and folders cannot be deleted / modified / copied / added whereas date and time cannot be changed by Analyst or Reviewer.
